# 💻 Shefoo – Personal Developer Portfolio

Welcome to the official repository of **Shefoo**, my personal developer portfolio built to showcase my skills, projects, and background as a front-end developer. This web app reflects my passion for clean UI, efficient code, and great user experiences.

## 🌐 Live Website

🔗 [shefoo.vercel.app](https://shefoo.vercel.app/)

## 📌 Features

- 🧑‍💻 About Me section with developer background and stack
- 🧩 Projects showcase with live links and GitHub repos
- 🧭 Smooth scroll and responsive layout for all devices
- 🌙 Dark theme aesthetic with modern visuals
- ⚡ Fast performance and optimized deployment

## 🛠️ Tech Stack

- **Framework**: Next.js (React)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Type Safety**: TypeScript
- **Hosting**: Vercel
