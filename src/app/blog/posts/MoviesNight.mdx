## 📝 About

To-Do List by SHEFO is a minimalist and responsive task management web application designed to help users efficiently organize their daily activities. The application offers a clean interface with essential functionalities, ensuring a seamless user experience across various devices.

🎬 About MoviesNight
MoviesNight is a sleek and modern movie discovery web app built for film lovers who want a quick and easy way to explore the world of cinema. From trending blockbusters to top-rated gems and upcoming releases, MoviesNight delivers a smooth and engaging experience for anyone looking to plan the perfect movie night.

Whether you're browsing casually or looking for something specific, MoviesNight helps you make that decision faster—with style.

##🚀 Features

🔥 Trending Movies: See what’s popular right now.

⭐ Top-Rated Picks: Explore critically acclaimed and fan-favorite movies.

🎉 Upcoming Releases: Stay updated with what's hitting theaters soon.

🖼️ Movie Details: Click any title to view high-quality posters, ratings, release dates, overviews, and more.

🔍 Responsive Design: Optimized for all screen sizes, so you can browse comfortably on desktop, tablet, or mobile.

⚡ Fast & Lightweight: Built with performance and simplicity in mind for a smooth user experience.

##🛠️ Tech Stack
React.js – For building the dynamic user interface.

TypeScript – Ensures type safety and better development experience.

Tailwind CSS – For rapid and responsive styling.

TMDB API – Movie data is fetched in real-time from The Movie Database (TMDB).

Vercel – Hosting and deployment.

##🧭 How to Use
Visit the App: Go to moviesnight-delta.vercel.app.

Browse Categories: Explore Trending, Top Rated, or Upcoming movies using the navigation bar.

Click a Movie: Get detailed information like overview, release date, average rating, and cover image.

Plan Your Night: Use the insights to pick the perfect movie for your night in!

##📌 Notes
This project is for educational and entertainment purposes.

Movie data is sourced live from TMDB and is regularly updated.

No account or login is needed—just open and explore.
