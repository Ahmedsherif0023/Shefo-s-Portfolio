## 📝 About

Velvet Pour is a visually rich and elegantly designed landing page created to showcase a luxury beverage brand. The page delivers a bold first impression with smooth transitions, atmospheric visuals, and a modern dark aesthetic. Designed to captivate users and reflect the sophistication of the product, the landing page emphasizes premium branding and immersive storytelling.

## 🚀 Features
• Hero Section: Striking visuals paired with smooth scroll-triggered animations.

• Storytelling Flow: A carefully structured layout guides users through the brand’s identity and experience.

• Parallax & Scroll Effects: Engaging animations using GSAP for seamless interactivity.

• Responsive Design: Optimized for both desktop and mobile viewing.

• Smooth Navigation: Sticky navbar with intuitive scroll behavior.

## 🛠️ Tech Stack
Frontend: React.js with Next.js for optimized performance and SEO.

Styling: Tailwind CSS for modern, responsive utility-based styling.

Animations: GSAP & ScrollTrigger for smooth scroll-based animations and effects.

Deployment: Vercel for fast, reliable deployment and CDN delivery.
